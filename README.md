# week8-database# COVID-19 Global Data Tracker

## Description
This project tracks COVID-19 data globally, including cases, deaths, and recoveries over time. The data is sourced from OWID (Our World In Data) and includes visualizations to help understand trends across various countries and regions.

## How to Run/Setup the Project
1. Clone the repository to your local machine:
2. Install necessary dependencies (e.g., `pandas`, `matplotlib`, etc.):
3. To run the project:
- Navigate to the project directory and run the main script:
  ```
  python covid_project.py
  ```
